# mineral-identification
Graph-based mineral identification project for Advanced Machine Learning (CS6365)

## Training Data
  Download the cleaned RRUFF images from the following Google Drive folder: ([`rruff_clean_images`](https://drive.google.com/drive/folders/1yt0P20HNioxjzQqK7P5-Eby3RoBOk4OH?usp=sharing))
  Access the LR-Raman files from the following Google Drive folder:([`LR-Raman`](https://drive.google.com/drive/folders/1DHclQDl9Qysb2Ve78d8689XZXmnPZ2Pv?usp=drive_link))

  The original mineral image and LR-Raman files are sourced from the [RRUFF Database](https://www.rruff.net/about/download-data/)
